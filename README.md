# Hepatitis-Prediction
Logistic Regression Model to predict the probability of a having hepatitis given various liver function test results such as albumin(ALB), alkaline phosphatase(ALP), alanine transaminase(ALT),	aspartate transaminase(AST), bilirubin(BIL), choline esterase(CHE), cholesterol (CHOL), creatine(CREA), gamma-glutamyl transferase (GGT) and total protein (PROT).
Categories found in the dataset include, '0=Blood Donor':0, '0s=suspect Blood Donor':0, '1=Hepatitis':1, '2=Fibrosis':2, '3=Cirrhosis':3.
